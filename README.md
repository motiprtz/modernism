# אפליקציית עזר למבחן - מודרניזם במוזיקה

אפליקציה מקיפה ללימוד מודרניזם במוזיקה, מבוססת על חומרי הקורס של פרופ' בלה ברובר-לובובסקי.

## 🎯 תכונות עיקריות

### 1. ציר הזמן (Timeline)
- סקירה כרונולוגית מלאה של כל האירועים, המלחינים והיצירות
- **חיפוש ומסננים**: חפש לפי מלחין, יצירה או אירוע
- **סינון לפי סוג**: לידות, פטירות, יצירות, בכורות, תנועות אמנותיות
- **פרופילי מלחינים**: הצג מידע מפורט על כל מלחין כולל שנות חיים, לאום, חשיבות ויצירות
- **תצוגה חזותית**: ציר זמן אינטראקטיבי עם אייקונים וצבעים לפי סוג אירוע

### 2. מפת מלחינים (Composer Map) 🆕
- **מפה אינטראקטיבית**: מפת עולם עם סמנים למיקומי לידת מלחינים
- **סינון לפי תקופה**: הצג מלחינים לפי ציר זמן (1800-1950)
- **לחיצה על ערים**: ראה את כל המלחינים שנולדו בכל עיר
- **פרטי מלחין**: לחץ על מלחין לקבלת מידע מפורט
- **תצוגה גיאוגרפית**: הבן את התפוצה הגיאוגרפית של המודרניזם

### 3. כרטיסיות מושגים (Flashcards)
- **{concepts.length} מושגים** חשובים ללימוד
- **מצב לימוד אינטראקטיבי**: הפוך כרטיסיות כדי לראות הגדרות
- **מעקב התקדמות**: סמן מושגים שנשלטו ועקוב אחר ההתקדמות שלך
- **מצבי לימוד**:
  - כל המושגים
  - מושגים שטרם נשלטו
  - מושגים שנשלטו
- **חיפוש ומסנן**: מצא מושגים ספציפיים במהירות
- **תצוגת רשימה**: ראה את כל המושגים בבת אחת
- **ערבוב**: למד במסדר אקראי

### 4. רשימת האזנה (Listening List) 🆕
- **מיון לפי מלחין**: כל היצירות מקובצות לפי מלחין
- **מיונים גמישים**: מיין לפי מלחין, כותרת או כרונולוגי
- **תצוגה מקובצת**: ראה את כל יצירות כל מלחין ביחד
- **חיפוש וסינון**: מצא יצירות ספציפיות במהירות
- **פירוט מלא**: כל יצירה עם מבנה ופרקים

### 5. יצירות מפורטות (Compositions)
- **{compositions.length} יצירות** מפורטות
- **ניתוח מבני מעמיק**: כל יצירה כוללת פירוט מלא של המבנה והמאפיינים
- **התייחסות נפרדת לפרקים**: יצירות מרובות פרקים מפורטות פרק אחר פרק
- **סינון לפי מלחין וז'אנר**: מצא יצירות ספציפיות בקלות
- **חיפוש**: חפש בכותרות, מלחינים ותוכן
- **הרחב/כווץ**: הצג או הסתר פרטים לפי הצורך

## 🚀 התחלת עבודה

### דרישות מקדימות
- Node.js (גרסה 16 ומעלה)
- npm או yarn

### התקנה

```bash
# התקן את התלויות
npm install

# הפעל את שרת הפיתוח
npm run dev

# בנה לפרודקשן
npm build
```

האפליקציה תרוץ על `http://localhost:3000/`

## 💡 טיפים ללימוד

1. **התחל בציר הזמן**: קבל תמונה כללית של התקופה, המלחינים והיצירות העיקריות
2. **למד את המושגים**: השתמש בכרטיסיות כדי לשלוט במושגים החשובים ובטרמינולוגיה
3. **הכר את היצירות**: עיין ברשימת היצירות והכר את המבנה והמאפיינים של כל יצירה
4. **חזור שוב ושוב**: השתמש במערכת המעקב כדי לוודא שאתה שולט בכל החומר
